# Protected Handler Local Visibility Commands

## Purpose
Lists the main local commands for inspecting protected-handler governance state.

## Commands

### Review exclusions
- `npm run review:protected-handlers`

### Run full verification
- `npm run verify`

### Future (planned)
- governance health summary (lightweight snapshot)

## Notes
- exclusion review output is derived from repo-state governance helpers
- full verification remains the authoritative integrity gate
- review reporting is for visibility, not for replacing verification
