# Protected Handler Governance Local Commands

## Purpose
Lists the main local commands for protected-handler governance visibility.

## Commands

### Full verification
- `npm run verify`

### Human-readable exclusion review
- `npm run review:protected-handlers`

### Machine-readable exclusion review
- `npm run review:protected-handlers:json`

### Governance health snapshot
- `npm run review:governance-health`

## Rule
Visibility commands support inspection. The verification gate remains the authoritative integrity check.
