# Protected Handler Governance Health Report Workflow

## Purpose
Defines how operators and maintainers inspect the current protected-handler governance health snapshot.

## Command
- `npm run review:governance-health`

## Output
- governed handler count
- excluded handler count
- near-due exclusions
- overdue exclusions

## When to Use
- before governance review work
- before cleanup of exclusions
- when checking current governance maturity and risk
